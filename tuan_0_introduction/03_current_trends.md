## 🎨 Frontend Frameworks — "The Big 4"

| Framework | Cha đẻ | Phù hợp cho | HotTrend |
|---|---|---|---|
| **React** ⭐ | Meta/Facebook | App lớn, team | 🔥🔥🔥🔥🔥 |
| **Vue.js** | Evan You | App nhỏ-vừa, solo | 🔥🔥🔥🔥 |
| **Angular** | Google | Enterprise | 🔥🔥🔥 |
| **Svelte** | Rich Harris | Performance-first | 🔥🔥 (rising) |

> **React** chiếm 65%+ job listings. **Vue.js** đang tăng trưởng mạnh ở Việt Nam (Shopee, VNG dùng Vue). **Angular** phổ biến ở banking, enterprise.

### Xu hướng 2025-2026:
- **Server Components** (React): Render trên server, gửi HTML cho client
- **Vite** thay thế Webpack: Build nhanh gấp 10-100 lần
- **TypeScript** đang trở thành BẮT BUỘC (không còn optional)

## ⚙️ Backend — Node.js, Python, Go

| | Node.js | Python | Go |
|---|---|---|---|
| **Điểm mạnh** | Cùng JS với FE | AI/ML, Data | Performance cực |
| **Framework** | Express, NestJS | Django, FastAPI | Gin, Echo |
| **Dùng cho** | Real-time, API | AI, Web app | Microservices |

### Xu hướng: 
- **FastAPI** (Python) đang bùng nổ — nhanh hơn Flask gấp 100 lần
- **Bun / Deno** — đe dọa vị trí Node.js

## ☁️ Cloud & AI — Tương lai Web

| Xu hướng | Ý nghĩa | Ví dụ |
|---|---|---|
| **Serverless** | Không quản lý server | Vercel, AWS Lambda |
| **Edge Computing** | Code chạy gần user | Cloudflare Workers |
| **AI Integration** | AI trong mọi web app | ChatGPT API, Copilot |
| **WebAssembly** | Code C/Rust chạy trên browser | Game, video editing |

> **Chị Hà:** *"Tại Shopee, AI đã auto-generate 30% code review. 5 năm nữa, developer không biết dùng AI = developer mất việc."*
