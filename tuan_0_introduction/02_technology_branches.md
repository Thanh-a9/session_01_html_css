## 🎬 "Frontend Hay Backend?" — Câu Hỏi Triệu Đô

*Minh vào nhóm CNTT khóa 2026. Câu hỏi đầu tiên: "Nên học Frontend hay Backend?"*

*Anh Hùng trả lời: "Sai câu hỏi. Câu hỏi đúng: Em thích TẠO THỨ NGƯỜI DÙNG THẤY (Frontend) hay TẠO THỨ NGƯỜI DÙNG KHÔNG THẤY NHƯNG MỌI THỨ PHỤ THUỘC VÀO NÓ (Backend)?"*

*"Giống xây nhà: Frontend = kiến trúc sư (thiết kế mặt tiền, nội thất). Backend = kỹ sư kết cấu (móng, cốt thép, điện nước). Thiếu một = sập."*

---

## 🎯 Mục tiêu
- Phân biệt Frontend, Backend, Full-stack, DevOps
- Biết công nghệ chính của từng nhánh
- Có định hướng chọn nhánh phù hợp

---

## 🎨 Frontend — "Mặt tiền cửa hàng"

**Bạn chịu trách nhiệm:** Giao diện, trải nghiệm, mọi thứ user nhìn thấy và chạm vào.

| Công nghệ | Vai trò |
|---|---|
| **HTML** | Cấu trúc (khung xương) |
| **CSS** | Styling (trang trí) |
| **JavaScript** | Logic (tương tác) |
| **React / Vue / Angular** | Framework (vũ khí nâng cao) |
| **Bootstrap / Tailwind** | CSS Framework (components sẵn) |

> ✅ **Phù hợp nếu:** Thích design, UI/UX, thấy kết quả ngay, visual thinking.

---

## ⚙️ Backend — "Hệ thống điện nước"

**Bạn chịu trách nhiệm:** Server, database, API, security — tất cả logic phía sau.

| Công nghệ | Vai trò |
|---|---|
| **Node.js** | JS phía server (dùng cùng ngôn ngữ FE) |
| **Python** | Dễ học, tốt cho AI/ML |
| **Java** | Enterprise, performance |
| **Go** | Microservices, concurrency |
| **MySQL / MongoDB** | Database (SQL / NoSQL) |

> ✅ **Phù hợp nếu:** Thích logic, algorithms, data, hệ thống.

---

## 🔄 Full-stack — "Cân hai vai"

| Stack | Frontend | Backend | Database |
|---|---|---|---|
| **MERN** | React | Express/Node.js | MongoDB |
| **MEAN** | Angular | Express/Node.js | MongoDB |
| **Django** | React/Vue | Django (Python) | PostgreSQL |

> **Chị Hà:** *"Full-stack không có nghĩa GIỎI cả hai. Mà là HIỂU cả hai, CHUYÊN một."*

---

## 🚀 DevOps — "Người đưa code lên production"

```
Code → Build → Test → Deploy → Monitor → Repeat
        ↑ DevOps quản lý toàn bộ pipeline này ↑
```

Docker, Kubernetes, AWS, GitHub Actions, CI/CD → tự động hóa mọi thứ.
