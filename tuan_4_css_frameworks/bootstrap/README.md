# 🟦 BOOTSTRAP 5 - CSS FRAMEWORK
# **Lộ trình học tập toàn diện**

Bootstrap là CSS Framework phổ biến nhất thế giới, giúp bạn xây dựng website responsive nhanh chóng với các component có sẵn.

### 03. Components
- `03_components.md` - Buttons, Cards, Navbar, Forms, Modals, và nhiều hơn nữa

### 04. Utilities
- `04_utilities.md` - Spacing, Colors, Typography, Display, Flexbox utilities

### 05. Customization
- `05_customization.md` - Customize Bootstrap với SASS variables, Override styles

## 🎯 MỤC TIÊU SAU KHI HOÀN THÀNH

Sau khi học xong lộ trình này, bạn sẽ:
- ✅ Hiểu Bootstrap là gì và khi nào nên dùng
- ✅ Cài đặt Bootstrap vào project (CDN và npm)
- ✅ Sử dụng Grid System để tạo layout responsive
- ✅ Sử dụng các Components phổ biến (Buttons, Cards, Navbar, Forms)
- ✅ Sử dụng Utilities để style nhanh
- ✅ Customize Bootstrap với SASS variables
